# DOCUMENTO 01 — Visión del Producto (condensado)

## Definición

**Hermes Command Center (HCC)** es un Sistema Operativo de Inteligencia Estratégica multimodal.
No es una app, ni un dashboard, ni un chatbot: es un **entorno operativo inteligente** donde el usuario
interactúa con un sistema que piensa, analiza y recomienda en tiempo real.

La **política es el primer vertical**. El mismo núcleo (Hermes Core) habilitará Hermes Government,
Corporate, Security, etc.

## Principio fundamental

> **La información no tiene valor si no mejora una decisión.**

- No se muestran datos sin interpretación.
- No se presentan métricas sin contexto.
- No se entregan respuestas sin recomendación.

## Los 5 Principios del Producto

1. **Invisible** — si el usuario necesita capacitación, fallamos.
2. **Predictivo** — Hermes no espera preguntas; las anticipa.
3. **Conversacional** — todo se hace hablando, no navegando menús.
4. **Visual** — la información se comprende en segundos.
5. **Estratégico** — cada respuesta ayuda a tomar una mejor decisión.

## Filosofía

No mostramos datos → **contamos historias**.
No hacemos reportes → **generamos decisiones**.
No respondemos preguntas → **resolvemos problemas**.
No construimos dashboards → **construimos inteligencia**.

## La Regla del Presidente

> Antes de desarrollar cualquier función: *"Si el Presidente de un país, un gobernador o el director
> de una gran empresa usara esto todos los días, ¿realmente le ayudaría a tomar una mejor decisión?"*
> Si la respuesta es no, no se desarrolla.

## Definición de éxito

Un usuario entra a la sala, activa Hermes por voz, y recibe **análisis + recomendaciones** útiles
sin tocar teclado ni mouse.
